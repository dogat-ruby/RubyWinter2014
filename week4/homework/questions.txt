Chapter 10 Basic Input and Output

1. How does Ruby read files?
Using the File class

2. How would you output "Hello World!" to a file called my_output.txt?

File.open('my_output.txt', 'w') do |f|
    f.puts "Hello World!"
end

3. What is the Directory class and what is it used for?
It is a class to manage directories.

4. What is an IO object?
It is an input output object to write to/read from a file. 

5. What is rake and what is it used for? What is a rake task?
It is a task management tool.  It allows you to specify tasks and describe dependencies as well as to group tasks in a namespace. Rake task is a set of code to be run on rake command.
